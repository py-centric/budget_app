# Financial Tools

## Overview

Financial tools provide calculators and utilities for various financial planning needs.

## Available Tools

### Tip Calculator

Calculate tips for services:
- Enter bill amount
- Select tip percentage
- Split among multiple people
- View per-person total

### Currency Converter

Convert between currencies:
- Select source currency
- Select target currency
- Enter amount
- View converted amount

### Loan Calculator

Calculate loan payments:
- Principal amount
- Interest rate
- Loan term
- Monthly payment calculation

### Savings Calculator

Project savings growth:
- Initial deposit
- Monthly contribution
- Interest rate
- Time period
- Final amount projection

## Accessing Tools

Navigate to the Tools Hub from the main navigation to access all financial calculators.
