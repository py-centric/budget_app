# Quickstart: Financial Calculators

## 1. Accessing Tools
1. Open the Navigation Drawer.
2. Select **Financial Tools**.
3. Verify you see the hub with options for:
    - Net Worth
    - Loan / Amortization
    - Savings / Compound Interest

## 2. Verify Net Worth
1. Select **Net Worth Calculator**.
2. Add an Asset: "Cash" -> $5,000.
3. Add a Liability: "Credit Card" -> $1,000.
4. Verify Net Worth displays **$4,000.00**.
5. Click **Save Snapshot**, enter "March 2026", and verify it appears in your saved list.

## 3. Verify Loan Calculator
1. Select **Loan Calculator**.
2. Enter:
    - Principal: $10,000
    - Interest: 12%
    - Term: 1 Year
3. Verify Monthly Payment is approximately **$888.49**.
4. Tap **View Schedule** and verify 12 months are listed.

## 4. Verify Savings Calculator
1. Select **Savings Calculator**.
2. Enter:
    - Initial: $1,000
    - Monthly: $100
    - Interest: 10%
    - Term: 5 Years
3. Verify Projected Total is approximately **$9,396.73**.

## 5. Persistence Check
1. Save any calculation.
2. Close the app and relaunch.
3. Navigate back to **Financial Tools** -> **Saved Items**.
4. Verify your calculation is still there and can be reloaded.
