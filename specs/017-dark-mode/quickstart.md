# Quickstart: Dark Mode Support

## 1. Verify Initial Setup
1. Open the app for the first time.
2. Verify the theme matches your device's system theme (Dark if OS is dark, Light if OS is light).

## 2. Manual Theme Selection
1. Navigate to **Settings**.
2. Find the **Theme Mode** dropdown.
3. Select **Dark**.
4. Verify the entire UI immediately changes to a dark background with light text.
5. Select **Light**.
6. Verify the UI reverts to the standard light theme.

## 3. Persistence Check
1. Set the app to **Dark** mode.
2. Close the app (kill the process).
3. Relaunch the app.
4. Verify the app starts in **Dark** mode immediately.

## 4. System Sync Check
1. In settings, select **System Default**.
2. Go to your device's OS settings and toggle Dark Mode.
3. Return to the app and verify it followed the change.
