# Quickstart: Home List Totals and Potential Balances

## 1. Verify Actual Totals
1. Ensure you have actual income and expense items for the current period.
2. Observe the "Income" and "Expenses" headers on the Home Page.
3. Verify they show the sum of the actual items in their respective lists.

## 2. Verify Potential Totals
1. Add a "Potential" income item (e.g., $500).
2. Observe the "Income" header.
3. Verify it now displays both the Actual total and the Potential total (e.g., "$1,000.00 | Pot: $1,500.00").
4. Repeat for "Expenses".

## 3. Verify Empty State/Zero Potential
1. Delete all potential items.
2. Verify the headers revert to showing only the single "Actual" total to save space.

## 4. Verify Update Performance
1. Add/Confirm/Delete an item.
2. Verify the header total updates instantly (no noticeable lag).
