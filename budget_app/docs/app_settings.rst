# App Settings

## Overview

Configure the app behavior and preferences to suit your needs.

## General Settings

### Currency

- Default currency selection
- Currency symbol display
- Decimal places

### Date Format

- Choose date format:
  - DD/MM/YYYY
  - MM/DD/YYYY
  - YYYY-MM-DD

### First Day of Week

- Sunday
- Monday

## App Lock

### PIN Protection

- Enable/disable app lock
- Set 4-digit PIN
- Require PIN on app launch

### Biometric Authentication

- Fingerprint unlock
- Face recognition (device-dependent)
- Fallback to PIN

## Categories

- Manage custom categories
- Add/edit/delete categories
- Choose icons and colors

## Recurring Transactions

- View all recurring transactions
- Enable/disable individual items

## Export & Backup

- Export data to CSV/PDF/Excel
- Create database backup
- Restore from backup

## Budget Reset

### Reset Options

- Clear current budget
- Clear all budgets
- Reset categories to default

### Data Reset

- Confirm before reset
- Option to keep categories
- Option to keep settings

- App version

- Privacy information
